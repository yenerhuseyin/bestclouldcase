## Installation
### Manuel Installation

1. Clone the repo
   ```sh
    git clone https://github.com/yenerhuseyin/bestclouldcase/
   ```
2. Build docker image
   ```sh
    docker build -t  yenerhuseyin/bestclouldcase .
   ```
3. Run container
   ```sh
    docker run -d -p 9000:80 yenerhuseyin/bestclouldcase/
   ```
4. Go to http://localhost:9000
